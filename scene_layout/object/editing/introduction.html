简介¶

本节介绍在物体模式下操作物体的工具。所有编辑工具都可以在物体菜单中找到。

合并¶

参考

模式

物体模式

菜单

物体‣ 合并

快捷键

Ctrl-J

合并将所有选定物体合并到上一个选定的 活动 物体中。所有物体数据都链接到活动物体(必须选择)。所有物体必须具有相同的类型：网格、曲线、曲面或骨架。如果并入了几条曲线，则每条曲线将保持其子类型(NURBS或贝塞尔)。

Note

物体数据有许多属性可以在加入时处理。

材质、顶点组、UV和顶点图层将被合并。

修改器、约束、组和父关系在并入时将被忽略，不会应用于活动物体。

转换为¶

网格/文本线->曲线¶

参考

模式

物体模式

菜单

物体(Object) ‣ 转换为(Convert to) ‣ 网格/文本线->曲线(Curve from Mesh/Text)

网格物体和文本物体可以转换为曲线物体。在网格物体中，仅考虑属于无面的边。所得到的曲线将是多段线曲线类型，但可以转换为具有平滑段，如 设置样条类型 中所述。

曲线/融球/曲面/文字转网格¶

参考

模式

物体模式

菜单

物体(Object) ‣ 转换为(Convert to) ‣ 曲线/融球/表面/文字转网格(Mesh from Curve/Meta/Surf/Text)

将选定的曲线，融球，曲面和文本物体转换为网格物体。这些物体的实际定义的分辨率将被考虑在转换中。请注意，它还可以保留由封闭和挤出曲线创建的面和体积。

曲线 -> 蜡笔¶

参考

模式

物体模式

菜单

物体(Object) ‣ 转换为(Convert to) ‣ 曲线 -> 蜡笔

将所选曲线转换为与曲线匹配的蜡笔物体；同时添加基础材质。选择多条曲线时，它们都将转换为同一个蜡笔物体。

选项¶

保留原物体

在转换之前复制原物体。

显示/隐藏¶

参考

模式

所有模式

菜单

物体(Object) ‣ 显示/隐藏(Show/Hide)

显示 隐藏物体 Alt-H

显示所有隐藏的物体。

隐藏选中项 H

隐藏所有选定的对象。

隐藏未选项 Shift-H

隐藏所未选定的对象。

删除¶

参考

模式

物体模式

菜单

物体(Object) ‣ 删除(Delete)

快捷键

X 或 Delete

从场景中删除所选对象。

全局删除

从所有场景中删除所选对象。
